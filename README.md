# OS/2 Warp 4
GTK theme based on the appearance of OS/2 Warp 4

![win-2.0](https://raw.githubusercontent.com/B00merang-Project/gallery/master/OS_2%20Warp%204%20(1).png)

**License :** GPL v3

**More info :** http://b00merang.weebly.com/os-2-warp-4.html

### Manual installation

Extract the zip file to the themes directory i.e. `/home/USERNAME/.themes`

### Requirements

- GTK+ 3.20 or above
- Murrine and Pixmap theme engines
